# AI Excellence Track

The AI Excellence Track Is A Structured Commitment Driven Learning Journey Designed To Guide Motivated Students From Gaza Toward Becoming Highly Capable Practitioners In Artificial Intelligence The Track Emphasizes

- Building Strong Foundations In Machine Learning And Deep Learning From Basics To Advanced Topics
- Developing Correct Thinking And Problem Solving Skills For AI Tasks With Solid Theoretical Understanding
- Gaining Practical Experience By Applying AI Concepts In Projects And Real World Problems
- Learning Independent Study Research Paper Reading And Continuous Self Improvement

This Track Extends The Machine Learning Course And Includes A Certification Based On The Stanford Machine Learning Curriculum

## Teaching Style And Evaluation
- Language Of Instruction English
- Focus On Practical Learning Especially Python Coding And Machine Learning Libraries
- Attendance Mandatory For All Lectures
- Evaluation Includes Online Practical Exam And Oral Exam To Assess Conceptual Understanding

## Academic Integrity
- Cheating Plagiarism Or Dishonesty Is Strictly Prohibited
- All Work Must Be Done Individually And Honestly
- Use Of AI Tools Or External Help Is Not Allowed Unless Explicitly Permitted By The Instructor

By Joining Students Commit To Discipline Consistency Ethical Conduct And Active Engagement
